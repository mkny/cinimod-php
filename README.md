Installation
============

## Laravel

Run `composer require mkny/cinimod:dev-master` to add the project to your composer.json

or add manually

```javascript
{
    "require": {
        "mkny/cinimod": "dev-master"
    }
}
```
Add the service provider in `app/config/app.php`.
```php
'Mkny\Cinimod\CinimodServiceProvider'
```
